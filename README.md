## 🚀 목적

- 함께 object 책을 읽고 이야기 나누고 싶은 주제를 각자 정하고, 토론한다.

## 📆 일정

- 요일: 금요일
- 시간: 17:00 ~ 18:00

## ⭐️ 진행 방식

1. 매 주 한 챕터 읽기
2. 깃허브 이슈 남기기(스터디 시작 직전까지 최소 1개)
    1. **SRP 원칙을 지킬 것**
    2. 질문
    3. 느낀점
    4. 토론거리
3. 이슈에 댓글 남기기(선택 사항)
4. 금요일에 대면으로 리마인드 및 추가 토론
5. 주말 동안 본인 이슈에 결론 짓기
